------------------------ ChadQuake-Makefile ------------------------
ChadQuake-Makefile is a fork of ChadQuake
(https://github.com/adolfintel/ChadQuake) with a Makefile build
system. This is only tested on Devuan Linux. It compiles with GCC
and G++. Tested only on Devuan Chimeara.
